## 作业2

通过多方查找完成了第三人称射击游戏的，像CS一样制作了ClientWithSession的联机模式

联机方式：玩家1点击创建游戏会自动进入游戏并创建会话，玩家2点击加入游戏会自动寻找并加入局域网中已存在的会话

操作方法：最多两个玩家，玩家2加入后游戏自动开始，WSAD移动，空格键跳跃，鼠标控制视角和开枪
